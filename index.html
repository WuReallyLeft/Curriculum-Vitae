這次實習經驗讓我更深深地了解了資訊管理領域的實際應用，並培養了我在解決問題、溝通和技術支援方面的能力。我期待將這些經驗和技能應用於未來的職業發展中，為組織和團隊帶來價值。

國立聯合大學 - 暑期培訓計畫

2023年 - 2024年

在國立聯合大學暑期計畫中，我同時擔任學生和老師的角色。這個計畫採取高年級學生教授低年級學生的模式，並由專業老師輔助，涵蓋前端、後端、資料庫、行動開發（Android）、人工智慧（AI）、物聯網（IoT）等領域的實作與經驗分享。

作為學生，我在這個暑期計畫中學習了程式設計的基礎知識，包括後端開發的基礎與進階技術、前端開發的基礎與進階技術、Android 應用開發，以及 ChatGPT API 的應用。

作為老師，我負責教授自己專題研究的內容，主要聚焦於 OpenAI API、LlamaIndex 及 RAG（檢索增強生成）的技術應用，幫助學員理解並實作這些技術。

這段經驗不僅讓我深化了自身的技術能力，也提升了教學與溝通能力，透過分享知識與實作練習，與學員共同成長。

國立聯合大學 - 程式設計助教

2024年 - 2025年

在擔任程式設計課程的助教期間，我的主要職責包括作業審查與學生指導。我們需要負責檢查這門課程的所有作業，並且要求學生當面展示他們的程式碼，確保其正確性與可讀性。這不僅需要具備細心的審查能力，也考驗我們的即時問題解決能力。

在審查過程中，我們不只是單純檢查程式是否能夠運行，而是進一步確認程式碼的結構、邏輯是否合理，以及是否符合最佳實踐（如變數命名、程式碼可讀性、效能優化等）。如果發現學生有理解上的盲點，或程式碼有邏輯錯誤，我們會現場進行指導，幫助學生釐清觀念，並引導他們思考如何改進程式碼，而不是直接給予答案。

這段助教經歷不僅強化了我的程式設計能力，也提升了我的溝通技巧與教學能力。面對不同背景與程度的學生，我學會如何用更清晰的方式解釋程式邏輯，並根據學生的理解程度調整教學方式。此外，這也讓我更加熟悉如何在團隊中協作，與其他助教共同維護課程品質，確保每位學生都能獲得良好的學習體驗。
